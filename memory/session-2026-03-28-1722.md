# Session 2026-03-28

## מה עשינו

### Open Graph — preview בווצאפ ורשתות חברתיות
- הוספנו route חדש `/song/<youtube_video_id>` ב-Flask
- ה-route קורא את ה-cache ומחזיר HTML עם OG meta tags (og:title, og:image, og:description, twitter:card)
- og:image = YouTube maxresdefault thumbnail
- og:description = "Letras by Ori Mosenzon"
- JS מזהה גם URL מסוג `/song/ID` (בנוסף ל-`/?v=ID`)
- `buildShareUrl` ו-`updateUrl` עברו לפורמט `/song/VIDEO_ID`

### Share dropdown
- כפתור Share עבר ל-dropdown menu עם אפשרויות:
  - 📋 Copy link
  - ⏱ Copy from here (עם timestamp)
  - 👍 Facebook
  - 💬 WhatsApp
  - 🐦 Twitter / X
  - 📲 Share via... (native OS sheet — מוצג רק כש-Web Share API זמין)
- תיקון z-index: top-bar הועלה ל-z-index 50 כדי שהdropdown יופיע מעל המילים

### Layout דו-עמודי על דסקטופ
- הוספנו `#main-content` wrapper סביב `#player-top` ו-`#lyrics`
- על מסכים >750px: הנגן (380px) משמאל, מילים מימין — הרבה יותר מקום למילים
- מובייל: layout קודם ללא שינוי

### תיקון performer שגוי (credits v6)
- באג: MusicBrainz "reversed search" החזיר performer שלא קשור לאמן ה-YouTube
- תיקון: אחרי קבלת performer מ-MusicBrainz, בודקים שהוא מתאים לאמן מ-YouTube title. אם לא — מעדיפים את YouTube artist
- `credits_version` הועלה ל-6 — cache backfill יריץ מחדש אוטומטית
